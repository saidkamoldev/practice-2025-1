Проект 1

Изучение русского языка
как иностранного

Куратор: Юрченко А.В.

Цель проекта — создание цифровой образовательной платформы, помогающей иностранным студентам эффективно изучать русский язык в интерактивном формате.

Платформа предоставляет структурированные учебные материалы, упражнения на грамматику и лексику, а также систему отслеживания прогресса обучающихся.

Роль Журакулова С.У. — Backend-разработчик и DevOps-инженер. Спроектирована и реализована полная серверная архитектура: REST API на FastAPI, база данных PostgreSQL, Docker-контейнеризация, система аутентификации JWT, деплой на Linux-сервере.

Роль Аширматова А.Х. — Frontend-разработчик. Реализован пользовательский интерфейс платформы: компоненты React, адаптивная вёрстка, интеграция с API, UX-логика учебного процесса.

Проект 2

Buddy — программа
поддержки студентов

Куратор: Косарин С.П.

Цель проекта — цифровая платформа программы «Buddy», помогающая иностранным студентам адаптироваться в новой языковой среде через взаимодействие с русскоязычными наставниками.

Платформа автоматизирует подбор пар «иностранный студент — наставник», предоставляет инструменты для общения, планирования встреч и отслеживания активности.

Задачи, выполненные командой: разработка компонентов React, проектирование UI/UX интерфейса, реализация форм регистрации и профилей, интеграция с бэкенд-сервисами через REST API.

C Text Editor
— Kilo

Реализация текстового редактора с нуля на языке C по проекту viewsourcecode.org/snaptoken/kilo

📖

Что такое Kilo?

Kilo — минималистичный текстовый редактор для терминала (~1000 строк кода на C), работающий в raw-режиме. Проект обучает низкоуровневому программированию: управлению терминалом, работе с файлами и обработке ввода.

🔧

Что реализовано

Raw-режим терминала, перемещение курсора через escape-последовательности, чтение и запись файлов, подсветка синтаксиса, поиск по тексту, команды редактирования (вставка, удаление строк).
